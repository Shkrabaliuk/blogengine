Цей пост показує всі можливості типографічної обробки тексту.

## Лапки

Звичайні лапки "автоматично" перетворюються на «ёлочки».

Вкладені лапки: «текст з "вкладеними лапками" всередині».

## Тире та дефіси

Подвійний дефіс -- перетворюється на довге тире.

Діапазони цифр: 2020-2024 стає 2020–2024.

Просто тире — з пробілами.

## Неразривний пробіл

Прийменники: в доме, на столі, за вікном, у лісі.

Одиниці виміру: 100 км, 5 кг, 25 грн, 2024 рік.

## Спецсимволи

Копірайт: (c) 2024
Товарний знак: MyBrand(tm)
Зареєстрована марка: Logo(r)

## Стрілки

Ліворуч <-- центр --> праворуч
Двостороння <-> стрілка

## Математичні символи

Нерівність: 5 != 3
Еквівалентність: A <=> B
Імплікація: A ==> B
Плюс-мінус: температура +/- 2 градуси

## Код

Інлайн код: `echo "Hello"` всередині тексту.

Блок коду:

```php
function hello($name) {
    return "Привіт, $name!";
}
```

```javascript
const greeting = (name) => {
    console.log(`Hello, ${name}!`);
};
```
